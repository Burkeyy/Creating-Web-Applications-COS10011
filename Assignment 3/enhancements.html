Enhancements

I have implemented a simple animation to the Navigation bar used on every page of the site. Although it doesn't add any functionality to the site, it offers a sleeker and more interesting look to the navigation bar. It works by first floating the header to the left and the Nav List to the right. The "header::after" code is then used to 'clear' after the header, otherwise the header disappears. This is because both the logo and nav bar are floating, leading to the header shrinking away. We then format the list items/nav bar by removing the bullet style on the lists, changing the font color/style, and changing the padding and margins to centre the list items in the nav bar properly.

We then use code like "nav a:hover" to add an affect when we hover the cursor over the list items. Then "nav a::before" is used to create the 5px high elements above the list items at the top of the view port. We then change the width to 0% to hide the elements until we hover over it. Using the "nav a:hover::before" and setting the width to 100% then allows for the 5px high elements to show when the list item is hovered over with the cursor. We finish off the animation by adding an ease-in transition of 240ms.
