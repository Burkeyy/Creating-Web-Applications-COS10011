Enhancements 2

I've implemented two enhancements in this assignment. If you look at the menu nav above, you'll notice the current page is highlighted in black. I use the window.location.pathname to retrieve the current path to the html page. I then use a substring method to retrieve the string between the last "/" and ".". I store all of this in a variable called "selectedMenuID". I then retrieve what's in this variable with getElementbyID(selectedMenuID), and store it in another variable called "SelectedMenu". Finally, whatever is stored in "SelectedMenu" gets assigned the class name of "activeItem" which gets referenced by the CSS file.

The second enhancement is a simple timer that begins when the user enters the Quiz page. It works by calling the "timer" function every second, which adds 1 to the current "time" variable. It displays the result in the span element id "timeDisplay" using ".textContent". Once the condition "time == 200" is met, the "timeDisplay.textContent" will now show "Times up!!", and the submit button will be removed by using "getElementbyID("submit").remove. I then clear the interval using the "clearinterval" method.
